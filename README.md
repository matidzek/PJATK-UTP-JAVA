## Assignments

### UTP2 – Concurrency

| Task | Description |
|------|-------------|
| StringTask | Simulates a long-running string concatenation task with states: `CREATED`, `RUNNING`, `ABORTED`, `READY`; supports `start()` and `abort()` |
| Swing Task List | GUI with `JList` showing running tasks as `Future`; supports querying state, cancelling, and viewing results |
| Towary | Two threads cooperating – thread A reads goods from `../Towary.txt`, thread B sums weights; both print progress to console |

### UTP3 – Internationalisation (S_INTERDB)

Travel agency receives offer files from contractors in various locales (`pl_PL`, `en_GB`, `de_DE`, …). Each line contains tab-separated: locale, country, departure date, return date, destination type, price, currency.

- `TravelData` – parses offer files; `getOffersDescriptionsList(locale, dateFormat)` returns offers translated and formatted for the target locale
- `Database` – stores all offers in a DB (`createDb()`), opens a `JTable` GUI with locale selector (`showGui()`)

### UTP4 – Streams & File I/O

| Task | Description |
|------|-------------|
| Anagrams | Finds all anagram groups from `unixdict.txt`; prints the group(s) with the most anagrams; **requires Java Streams** |
| Finder | Reads `~/Test.java` and counts `if` statements and occurrences of `"wariant"` – ignoring comments and string literals |
| Futil | Recursively walks `~/UTP6dir/`, reads all `.txt` files (CP1250) and writes merged content to `UTP6res.txt` (UTF-8) using `FileVisitor` |

### UTP5 – Collections & Maps

| Task | Description |
|------|-------------|
| XList | Extended `ArrayList` with `union`, `diff`, `unique`, `combine`, `collect`, `join`, `forEachWithIndex`, and factory methods `charsOf` / `tokensOf` |
| CustomersPurchaseSortFind | Reads `~/customers.txt`; sorts purchases by surname and by cost; filters by customer ID |
| ProgLang | Reads `~/Programmers.tsv`; builds language→programmers and programmer→languages maps with sorting and filtering via generic `sorted()` / `filtered()` helpers |

### UTP6 – Lambdas & Functional Patterns

| Task | Description |
|------|-------------|
| Maybe | Null-safe container (`Maybe<T>`) with `of`, `ifPresent`, `map`, `get`, `isPresent`, `orElse`, `filter` |
| InputConverter | Generic converter with a single `convertBy(functions...)` method for composing arbitrary chains of `Function`s |
